90° Herringbone Pattern (from inchcalculator.com reference)

The 90° herringbone pattern: Each tile's SHORT edge aligns with the LONG edge of the adjacent tile.

Repeating unit (4 tiles):
┌───────────┬───┐
│     H1    │ V1│    H1 = Horizontal tile (L × W)
│           │ V1│    V1 = Vertical tile (W × L)
├───┬───────┴───┤
│ V2│     H2    │    V2 = Vertical tile (W × L)
│ V2│           │    H2 = Horizontal tile (L × W)
└───┴───────────┘

Each row is offset by (L + grout) to create the stepped pattern.
